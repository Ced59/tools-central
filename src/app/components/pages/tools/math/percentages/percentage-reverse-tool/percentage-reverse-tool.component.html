Définition & méthode

On inverse l’opération en divisant par le coefficient appliqué.

  1. Calculez le coefficient : 1 + X/100
  2. Divisez : Final ÷ coefficient
  3. Le résultat est la valeur initiale
Si X = -100%, le coefficient vaut 0 : on ne peut pas revenir en arrière.
