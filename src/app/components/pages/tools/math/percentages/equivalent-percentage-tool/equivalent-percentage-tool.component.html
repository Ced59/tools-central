Définition & méthode

Une suite de pourcentages se compose : on multiplie des coefficients.

  1. Convertissez chaque pourcentage en coefficient : 1 + p/100.
  2. Multipliez tous les coefficients pour obtenir le coefficient total.
  3. Transformez ce coefficient en un taux unique : (coef − 1) × 100.
Exemple : +10% puis +10% → coef = 1,1×1,1 = 1,21 → taux équivalent = +21%.
