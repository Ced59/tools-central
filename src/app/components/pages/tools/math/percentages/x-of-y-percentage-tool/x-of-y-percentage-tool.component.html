Définition & méthode

Cet outil calcule la part de X dans Y (en pourcentage).

  1. Calculez le ratio : X ÷ Y
  2. Multipliez par 100 pour obtenir un pourcentage
  3. Interprétez : 25% = 25 sur 100 (un quart)
Si Y = 0, le pourcentage n’est pas défini (division par 0). Si X > Y, le résultat peut dépasser 100%.
