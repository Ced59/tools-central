Définition & méthode

“A% de B% de C%” revient à multiplier A/100, B/100, C/100.

  1. Convertissez chaque niveau en fraction : p/100.
  2. Multipliez toutes les fractions pour obtenir le coefficient total.
  3. Multipliez le coefficient par 100 pour retrouver le pourcentage global.
