Définition & méthode

Une part relative compare une valeur à la somme de toutes les valeurs.

  1. Additionnez toutes les valeurs : total = A + B + C + ….
  2. Calculez le ratio : A ÷ total.
  3. Convertissez en pourcentage : (A ÷ total) × 100.
Si la somme vaut 0, les pourcentages ne sont pas définis.
