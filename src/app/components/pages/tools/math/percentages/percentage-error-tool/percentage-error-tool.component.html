Définition & méthode

Le pourcentage d’erreur mesure l’écart d’une valeur mesurée par rapport à une valeur de référence.

  1. Calculez l’écart : mesuré − référence.
  2. Prenez la valeur absolue (on mesure une amplitude d’erreur) : |…|.
  3. Divisez par la référence puis multipliez par 100.
À ne pas confondre : • “Variation en %” compare une valeur initiale à une valeur finale (référence = initiale). • “Écart relatif” peut choisir une référence (A ou B). • “Pourcentage d’erreur” impose la référence (valeur vraie/théorique) et utilise une valeur absolue. Exemple : référence 100, mesuré 102 ou 98 → erreur = 2% dans les deux cas.
Lecture rapide : mesuré / référence = {{ fmt(ratio()) }}. Si ce ratio vaut 1, la mesure est exacte.
