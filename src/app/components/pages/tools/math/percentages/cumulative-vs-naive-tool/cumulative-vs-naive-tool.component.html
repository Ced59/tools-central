Définition & méthode

Additionner des pourcentages est souvent faux : l’effet réel est composé.

  1. Transformez chaque pourcentage en coefficient : 1 + p/100.
  2. Multipliez les coefficients pour obtenir l’effet cumulé.
  3. Comparez avec l’addition naïve : p1 + p2.
Exemple : +10% puis +10% → cumulé = ×1,1×1,1 = ×1,21 (+21%), pas +20%.
