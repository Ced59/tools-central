Définition & méthode

En proportionnalité inverse, le produit reste constant.

  1. Calculez le produit : a · b.
  2. Posez l’égalité : a · b = c · x.
  3. Isolez x : x = (a · b) / c.
Si c augmente, x diminue (et inversement) : c × x reste constant.
