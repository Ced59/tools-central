Définition & formule

Proportion directe : le rapport b/a est constant.

  1. Calculez le ratio : b / a.
  2. Multipliez par c : x = (b/a) × c.
  3. Vérifiez que a ≠ 0.
Si c double, alors x double (proportion directe).
